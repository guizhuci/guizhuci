## 🚀 30分钟快速上线路线图

### 第一步：环境准备（5分钟）

```bash
# 1. 检查 Node.js 版本
node --version  # 需要 v18+

# 2. 安装 pnpm（如果没有）
npm install -g pnpm

# 3. 安装依赖
pnpm install

# 4. 创建环境变量文件
cp .env.example .env.local
# 编辑 .env.local，设置 DATABASE_URL
```

### 第二步：题库准备（10分钟）

**快速准备测试题库**：
1. 准备一个包含 10-20 道题的 Word 文档
2. 登录后台管理系统（/admin）
3. 进入"题库管理" -> "导入题库"
4. 选择科目和文档，点击导入

**题库格式要求**（查看 [QUESTION_IMPORT_GUIDE.md](./QUESTION_IMPORT_GUIDE.md)）：
```
1. 以下关于数据的描述，错误的是：
A. 数据可以存储在多种载体中
B. 数据是信息的载体
C. 数据只能以数字形式存在
D. 数据具有可处理性
答案：D
解析：数据可以以多种形式存在，不仅仅是数字。
```

### 第三步：生成VIP码（5分钟）

1. 登录后台管理系统
2. 进入"VIP码管理"
3. 点击"生成VIP码"
4. 生成 10 个 7 天测试码 + 50 个 7 天活动码

### 第四步：快速部署（10分钟）

**选择方案A：Vercel（推荐，最快）**
```bash
# 安装 Vercel CLI
pnpm add -g vercel

# 登录并部署
vercel login
vercel --prod
```

**选择方案B：本地运行（测试）**
```bash
# 使用快速部署脚本
bash scripts/quick-deploy.sh

# 或手动运行
pnpm run dev
```

### 第五步：上线验证（5分钟）

1. 访问部署后的网站
2. 测试核心功能：
   - [ ] 登录注册
   - [ ] 每日免费刷题
   - [ ] 专项练习
   - [ ] VIP码激活
3. 导出VIP码列表
4. 准备推广物料
