```sql
FROM subjects s
LEFT JOIN questions q ON s.id = q.subject_id
WHERE s.category = 'single_recruitment'
GROUP BY s.id, s.name
ORDER BY s.sort_order;
```

## 🚀 二、如何开始刷题

### 步骤1：访问网站

1. 打开浏览器，访问：http://localhost:5000
2. 点击"微信一键登录"按钮（模拟登录）
3. 登录成功后进入刷题界面

### 步骤2：选择刷题模式

登录后，你会看到4个刷题入口：

| 模式 | 说明 | 免费额度 | 适用场景 |
|------|------|---------|---------|
| 免费刷题 | 每日5道随机题 | 5题/天 | 轻松练习 |
| 专项练习 | 按科目练习 | 前9题免费 | 系统学习 |
| 全真模拟 | 模拟考试场景 | 部分免费 | 考前冲刺 |
| 错题集 | 复习错题 | 无限制 | 查漏补缺 |

### 模式2：专项练习（按科目）

**特点**：
- ✅ 按科目系统练习
- ✅ 前9题免费
- ✅ 第10题提示解锁（需要VIP码）
- ✅ 支持答题模式和背题模式

**刷题步骤**：

#### 2.1 选择科目
1. 点击"专项练习"按钮
2. 选择你要练习的科目：
   - 信息技术
   - 通用技术
   - 美术
   - 音乐
   - 综合实践

#### 2.2 选择练习模式
选择科目后，可以选择两种模式：

**答题模式**（默认）：
- 自己作答
- 答完显示对错和解析
- 适合自测

**背题模式**：
- 直接显示正确答案
- 边看边背
- 适合记忆

#### 2.3 开始刷题
- 答题模式：点击选项作答，提交后查看结果
- 背题模式：直接显示答案，记住即可

#### 2.4 第10题解锁
- 前9题免费练习
- 第10题会出现解锁提示

## 🔧 五、如何上传题目？

### 方法1：通过后台管理系统（推荐）

1. 访问 http://localhost:5000/admin
2. 输入管理员账号（默认：admin/admin）
3. 进入"题库管理"
4. 点击"导入题库"
5. 选择科目和Word文档
6. 点击"开始导入"

### 方法2：批量导入

如果你有多个文档，可以：
1. 准备好所有题库文档
2. 按科目分类
3. 依次导入每个文档

### 题库格式要求

详细格式请查看：[docs/QUESTION_IMPORT_GUIDE.md](./QUESTION_IMPORT_GUIDE.md)

**简单格式示例**：
```
信息技术
================================

1. 以下关于数据的描述，错误的是：
A. 数据可以存储在多种载体中
B. 数据是信息的载体
C. 数据只能以数字形式存在
D. 数据具有可处理性
答案：D
解析：数据可以以多种形式存在。

2. 信息的基本特征不包括：
A. 时效性
B. 可共享性
C. 客观性
D. 不可传递性
答案：D
解析：客观性不是信息的基本特征。
```

## 📊 六、如何查看答题记录？

### 查看答题统计

**方法1：错题集**
- 进入"错题集"
- 查看所有错题
- 按科目筛选

**方法2：后台统计**
- 进入后台管理系统
- 查看"用户答题记录"
- 导出统计数据

**方法3：数据库查询**
```sql
-- 查看自己的答题统计
SELECT
  DATE(created_at) as date,
  COUNT(*) as answer_count,
  SUM(CASE WHEN is_correct THEN 1 ELSE 0 END) as correct_count
FROM answer_records
WHERE user_id = '你的用户ID'
GROUP BY DATE(created_at)
ORDER BY date DESC;
```

## ❓ 七、常见问题

### Q1: 刷题是否需要注册？

**A**: 需要。点击"微信一键登录"模拟注册即可，无需真实信息。

### Q2: 如何获取VIP码？

**A**:
1. 测试期：联系客服获取测试码
2. 活动期：关注活动，免费领码
3. 购买：联系客服购买

### Q3: 每日免费题什么时候刷新？

**A**: 每天凌晨0点自动刷新，重新随机5道题。

### Q4: 错题多久更新一次？

**A**: 实时更新。答错的题目立即加入错题集。

### Q5: 可以跳题吗？

**A**: 可以。
- 免费刷题：必须按顺序答完
- 专项练习：可以跳过，但会记录为未答
- 全真模拟：可以自由切换题目

### Q6: 刷题记录会丢失吗？

**A**: 不会。所有答题记录都保存在数据库中，永久保存。
