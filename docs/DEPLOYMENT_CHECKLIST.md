## 🗄️ 二、数据库准备

### 2.1 基础数据
```sql
-- 检查科目数据
SELECT * FROM subjects WHERE category = 'single_recruitment';

-- 检查题目数量
SELECT s.name, COUNT(q.id) as question_count
FROM subjects s
LEFT JOIN questions q ON s.id = q.subject_id
WHERE s.category = 'single_recruitment'
GROUP BY s.id, s.name;

-- 检查模拟卷数据
SELECT * FROM exam_papers;
```

## ⚙️ 三、生产环境配置

### 3.1 环境变量配置
```env
# .env.production
DATABASE_URL=你的生产数据库连接字符串
NEXT_PUBLIC_APP_NAME=桂职测助手
NEXT_PUBLIC_APP_DESCRIPTION=广西单招刷题神器
```

### 3.2 部署平台选择

#### 方案A：Vercel（推荐，快速部署）
```bash
# 1. 安装 Vercel CLI
pnpm add -g vercel

# 2. 登录 Vercel
vercel login

# 3. 部署
vercel --prod
```

**优点**：
- 零配置部署
- 自动HTTPS
- 全球CDN
- 免费额度：100GB带宽/月

#### 方案B：阿里云/腾讯云（适合国内）
- 购买轻量应用服务器（2核4G，约50-100元/月）
- 安装 Node.js + PostgreSQL
- 使用 PM2 守护进程
- 配置 Nginx 反向代理

### 3.3 数据库选择

#### 方案A：Supabase（推荐）
- 免费额度：500MB数据库
- 内置PostgreSQL
- 可视化管理界面

#### 方案B：云数据库
- 阿里云RDS PostgreSQL
- 腾讯云PostgreSQL
- 预估费用：50-200元/月（根据配置）
