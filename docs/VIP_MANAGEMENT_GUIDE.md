# VIP码生成与管理指南

## 🎁 VIP码系统介绍

VIP码是用户解锁高级功能的唯一凭证，用户通过输入VIP码激活VIP权限，即可享受：
- ✅ 无限制刷题（不限科目、不限题量）
- ✅ 错题集完整功能
- ✅ 全真模拟卷（包括收费试卷）
- ✅ 答题模式和背题模式

---

## 📋 VIP码类型

### 1. 测试码（7天）
- 用途：内部测试、体验
- 数量：10-20个
- 有效期：7天
- 分发：仅限内部人员

### 2. 活动码（7-30天）
- 用途：推广活动、拉新
- 数量：50-100个
- 有效期：7天/30天
- 分发：活动参与者

### 3. 付费码（30天-365天）
- 用途：正式销售
- 数量：不限制
- 有效期：30天/90天/365天
- 分发：付费用户

## 🚀 快速生成VIP码

### 方法1：通过后台管理系统（推荐）

1. 登录后台管理系统
2. 进入"VIP码管理"页面
3. 点击"生成VIP码"按钮
4. 填写生成参数：
   - 生成数量：10
   - 有效天数：7
   - 批次说明：测试批-20250104
5. 点击"生成"
6. 保存生成的VIP码列表

### 方法2：通过数据库直接生成

```sql
-- 生成10个7天VIP码
INSERT INTO vip_codes (code, expire_date, batch_name, status)
SELECT
  'VIP' || LPAD(SEQUENCE.NEXTVAL, 6, '0'),  -- VIP码格式：VIP000001
  CURRENT_DATE + INTERVAL '7 days',        -- 7天后过期
  '测试批-' || TO_CHAR(CURRENT_DATE, 'YYYYMMDD'),
  'unused'
FROM GENERATE_SERIES(1, 10);
```

## 📊 销售数据管理

### 数据统计表

```sql
-- VIP码使用统计
SELECT
  DATE(created_at) as date,
  COUNT(*) as total_generated,
  SUM(CASE WHEN status = 'used' THEN 1 ELSE 0 END) as total_used,
  SUM(CASE WHEN status = 'unused' THEN 1 ELSE 0 END) as total_unused,
  ROUND(SUM(CASE WHEN status = 'used' THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) as usage_rate
FROM vip_codes
GROUP BY DATE(created_at)
ORDER BY date DESC;

-- VIP码到期统计
SELECT
  batch_name,
  COUNT(*) as total,
  SUM(CASE WHEN expire_date < CURRENT_DATE THEN 1 ELSE 0 END) as expired,
  SUM(CASE WHEN expire_date BETWEEN CURRENT_DATE AND CURRENT_DATE + INTERVAL '7 days' THEN 1 ELSE 0 END) as expiring_soon
FROM vip_codes
WHERE status = 'used'
GROUP BY batch_name;

-- 收入统计（假设每张VIP码售价39元）
SELECT
  DATE(used_at) as date,
  COUNT(*) as codes_used,
  COUNT(*) * 39 as revenue
FROM vip_codes
WHERE status = 'used'
GROUP BY DATE(used_at)
ORDER BY date DESC;
```

## 🔐 VIP码安全

### 安全措施

1. **VIP码格式复杂**
   - 建议：VIP + 8位随机数字
   - 示例：VIP83295103

2. **使用限制**
   - 每个VIP码只能使用一次
   - 使用后自动失效
   - 过期后无法激活

3. **防刷机制**
   - 同一IP地址每日最多激活3次
   - 同一设备每日最多激活1次
   - 异常激活需要人工审核

4. **监控告警**
   - 异常激活（短时间内大量激活）
   - 批量使用检测
   - 盗刷行为识别

## 📈 数据监控

### 核心指标

```typescript
// VIP激活率
activation_rate = (已激活VIP数 / 注册用户数) × 100%

// VIP续费率
renewal_rate = (续费VIP用户 / 到期VIP用户) × 100%

// VIP收入
daily_revenue = 当日激活VIP数 × 单价

// 平均客单价
avg_price = 总收入 / 激活VIP总数

// 转化漏斗
funnel = {
  visit: 访问人数,
  register: 注册人数,
  free_trial: 试用人数,
  vip_activate: VIP激活人数
}
```

### 监控建议

**每日监控**：
- 新增注册用户
- VIP激活数量
- 激活率
- 收入

**每周监控**：
- VIP续费率
- 平均客单价
- 渠道效果

**每月监控**：
- 用户留存率
- VIP使用时长
- 题库使用情况
