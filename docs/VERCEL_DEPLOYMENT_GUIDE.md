## 为什么"链接打不开"？

### 说明
当前应用运行在**沙箱本地环境**中，这意味着：
- ✅ 沙箱内部可以访问 `http://localhost:5000`
- ❌ 外部浏览器无法直接访问（因为这是沙箱内部地址）
- ❌ 您在其他电脑/手机上无法访问

### 解决方案
需要将应用部署到**公网服务器**，才能让所有人访问。

---

## Vercel 部署方案（推荐，完全免费）

### 特点
- ✅ **完全免费**
- ✅ **自动分配域名**（如 `guizhuci.vercel.app`）
- ✅ **无需注册域名**（可选绑定自定义域名）
- ✅ **自动 HTTPS**
- ✅ **全球 CDN 加速**
- ✅ **自动部署**（代码推送后自动更新）

### 部署步骤

#### 1. 安装 Vercel CLI
```bash
pnpm add -g vercel
```

#### 2. 登录 Vercel
```bash
vercel login
```
会打开浏览器，选择 GitHub/Google/GitLab 账号登录。

#### 3. 执行部署
```bash
vercel
```

按照提示操作：
- 是否链接到现有项目？选择 `No`
- 项目名称？输入 `guizhuci`（或其他）
- 部署目标？选择 `Production` 或 `Preview` 都可以

#### 4. 等待部署完成
部署完成后，会显示访问地址：
```
✅ Production: https://guizhuci.vercel.app
```

这个地址就是您的**公网访问地址**！

#### 5. 后续更新
每次修改代码后，再次执行：
```bash
vercel --prod
```

## 关于域名问题

### ❌ 不需要注册域名
Vercel 会自动分配免费子域名，格式为：
- `https://guizhuci.vercel.app`
- `https://your-app-name.vercel.app`

### 🎯 可选：绑定自定义域名
如果您有域名，可以绑定到 Vercel：
1. 登录 Vercel 控制台
2. 选择项目 → Settings → Domains
3. 添加您的域名（如 `guizhuci.com`）
4. 按照提示配置 DNS

**费用说明**：
- 使用 `.vercel.app` 免费子域名：**完全免费**
- 绑定自定义域名：Vercel 免费，但域名需要您自己购买（约 ¥60-100/年）

如果不想用 Vercel，也可以选择：

```
http://guizhuci.com → http://localhost:5000
```

## 常见问题

### Q1: Vercel 真的免费吗？
A: 是的，Vercel 有慷慨的免费套餐，足够小型项目使用。包括：
- 无限带宽
- 100GB 构建/月
- 自动 HTTPS
- 全球 CDN

### Q2: 我的数据安全吗？
A: Vercel 是知名的部署平台，安全性很高。但建议您：
- 使用环境变量存储敏感信息（如数据库密码）
- 定期备份数据库

### Q3: 部署后可以访问后台吗？
A: 可以！部署后访问 `https://guizhuci.vercel.app/admin` 即可进入后台。

### Q4: 部署后数据库怎么配置？
A: 需要使用云数据库（如 Supabase、Neon），在 Vercel 项目中配置环境变量 `DATABASE_URL`。
