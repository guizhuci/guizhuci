# 题库快速导入指南

## 📝 题库格式要求

### 格式一：题目在前，答案在后（推荐）

```markdown
信息技术（科目名）
================================

1. 以下关于数据的描述，错误的是：
A. 数据可以存储在多种载体中
B. 数据是信息的载体
C. 数据只能以数字形式存在
D. 数据具有可处理性
答案：D
解析：数据可以以数字、文字、图像、声音等多种形式存在。

2. 信息的基本特征不包括：
A. 时效性
B. 可共享性
C. 客观性
D. 不可传递性
答案：B
解析：信息的基本特征包括：时效性、可共享性、价值性、载体依附性、可传递性、可伪性等，但不包括客观性。

3. 天气预报中的"明天降雨概率为80%"体现了信息的：
A. 载体依附性
B. 时效性
C. 真伪性
D. 价值性
答案：D
解析：天气预报可以帮助人们决策，体现了信息的价值性。

4. 信息可以脱离载体独立存在。（ ）
答案：B
解析：信息必须依附于载体，不能脱离载体独立存在。

通用技术（科目名）
================================

1. 技术的性质包括：
A. 目的性、创新性、综合性、两面性、专利性
B. 目的性、实用性、经济性
C. 实用性、创新性、综合性
D. 目的性、创新性、综合性
答案：A
解析：技术的性质包括目的性、创新性、综合性、两面性、专利性。

2. 以下属于技术试验的是：
A. 阅读技术说明书
B. 检查产品外观
C. 测试汽车刹车性能
D. 学习技术原理
答案：C
解析：技术试验是检验技术性能的重要手段，测试汽车刹车性能属于技术试验。
```

## 🚀 快速导入步骤

### 步骤1：准备题库文档

**要求**：
- 文档格式：Word (.docx) 或 TXT (.txt)
- 文件大小：建议不超过 10MB
- 题目数量：每次导入建议不超过 200 题

### 步骤2：访问后台管理系统

1. 打开网站首页
2. 点击底部的"后台管理系统"链接
3. 输入管理员账号密码（默认：admin/admin）

### 步骤3：创建科目（如果需要）

1. 进入"科目管理"
2. 点击"添加科目"
3. 填写科目信息：
   - 科目名称：信息技术
   - 科目分类：单招考试
   - 排序序号：1
4. 点击"保存"

### 步骤4：导入题库

1. 进入"题库管理"
2. 点击"导入题库"按钮
3. 选择要导入的 Word 文档
4. 选择目标科目
5. 点击"开始导入"
6. 等待导入完成

### 步骤5：检查导入结果

导入完成后，系统会显示：
- 成功导入的题目数量
- 导入失败的题目数量（如果有）
- 失败原因

## ⚠️ 常见问题和解决方案

### 问题1：导入失败，提示"无法解析题目"

**原因**：
- 题目格式不符合要求
- 题目和答案之间缺少分隔符
- 选项格式混乱

**解决方法**：
- 按照标准格式调整题目
- 确保每道题目有明确的"答案："标记
- 选择题选项必须是 A、B、C、D 开头

### 问题2：判断题导入后变成选择题

**原因**：
- 判断题格式不规范
- 没有使用"（ ）"标识

**解决方法**：
- 在判断题题干后添加"（ ）"标识
- 例如："信息可以脱离载体独立存在。（ ）"

### 问题3：答案字段包含"参考答案："前缀

**原因**：
- 原题库中答案格式不统一

**解决方法**：
- API 已自动处理此问题
- 如果仍有问题，检查 answer_records 表

### 问题4：题目乱码

**原因**：
- Word 文档编码问题
- 特殊字符未正确处理

**解决方法**：
- 使用 UTF-8 编码保存 TXT 文件
- 简化题目内容，避免特殊符号

## 🔧 进阶技巧

### 技巧1：批量导入多个文档

如果你有多个题库文档：
1. 先导入第一个文档
2. 检查导入结果
3. 确认无误后，再导入第二个文档
4. 依次完成所有文档导入

### 技巧2：定期更新题库

建议每两周更新一次题库：
- 新增 10-20 道题目
- 替换过时题目
- 根据用户反馈优化题目

### 技巧3：题库分类管理

可以在科目下再细分类别：
- 信息技术 -> 理论题 / 应用题
- 通用技术 -> 设计题 / 制作题
- 美术 -> 理论题 / 鉴赏题
