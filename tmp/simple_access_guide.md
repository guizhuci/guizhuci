# 快速访问指南 - 请选择你的情况

## 📋 请告诉我：你是如何访问这个开发环境的？

### 情况1：我是通过 SSH 终端连接的

**最简单的方法：**

在**本地电脑的终端**（不是SSH连接的终端）中执行：
```bash
ssh -L 5000:localhost:5000 root@服务器IP
```

然后在浏览器访问：http://localhost:5000/admin

**如果不知道服务器IP，在SSH终端执行：**
```bash
hostname -I | awk '{print $1}'
```

---

### 情况2：我是通过 VS Code Remote SSH 连接

**方法A：使用简单浏览器扩展**
1. 在 VS Code 中按 Ctrl+Shift+X 打开扩展
2. 搜索 "Simple Browser" 并安装
3. 右键点击 → 选择 "Simple Browser: Show"
4. 输入 http://localhost:5000/admin

**方法B：配置端口转发**
1. 按 F1 打开命令面板
2. 输入 "Forward a Port"
3. 输入 5000
4. 点击弹出链接

---

### 情况3：我是通过 Coze/Coder 平台

**查找预览按钮：**
1. 查看页面顶部或右侧
2. 寻找"预览" (Preview) 或 "打开" 按钮
3. 点击后会自动在浏览器打开
4. 平台会自动配置好访问

**查看控制台：**
- 查看终端输出，可能会有预览URL
- 类似：https://xxxx.preview.coze.site

---

### 情况4：我是通过网页IDE（如 Gitpod, Replit 等）

**查找预览按钮：**
- 通常在页面右上角
- 图标是 "眼睛" 或 "打开外部链接"
- 点击后会生成一个临时URL

---

### 情况5：我在本地电脑上开发

**直接访问：**
```
http://localhost:5000/admin
```

如果不行，尝试：
```
http://127.0.0.1:5000/admin
```

---

## 🔍 如何确认？

**在SSH终端执行以下命令：**
```bash
curl http://localhost:5000/admin | grep "后台管理系统"
```

如果输出"后台管理系统"，说明服务正常，只是访问方式问题。

## 💡 最可能的解决方案

如果你在**云平台或网页IDE**中：
→ 找"预览"按钮，点击即可

如果你通过**SSH连接**：
→ 需要配置端口转发（见情况1）
