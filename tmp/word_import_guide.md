# Word题库导入测试说明

## 功能概述
桂职测助手支持从Word文档批量导入题目，智能识别题目与答案分离的格式。

## 支持的题型
- 单项选择题
- 判断题
- 综合探究题

## 文档格式要求

### 1. 单项选择题格式
```
1. 题目内容
A. 选项A内容
B. 选项B内容
C. 选项C内容
D. 选项D内容
```

### 2. 判断题格式
```
1. 题目内容
```

### 3. 综合探究题格式
```
1. 题目内容
```

## 答案格式要求

### 参考答案部分
在文档末尾添加"参考答案"作为分隔符，然后按题型组织答案：

### 单项选择题答案格式
```
参考答案
单项选择题
1. A【答案解析内容】
2. B【答案解析内容】
```

### 判断题答案格式
```
判断题
1. 正确【答案解析内容】
2. 错误【答案解析内容】
```

### 综合探究题答案格式
```
综合探究题
1. 答：参考答案内容
```

## 完整示例

```
单项选择题
1. 下列哪项是正确的？
A. 选项A
B. 选项B
C. 选项C
D. 选项D
2. 题目内容？
A. 选项A
B. 选项B
C. 选项C
D. 选项D

判断题
1. 这是一个判断题
2. 这是另一个判断题

综合探究题
1. 这是一个综合探究题

参考答案
单项选择题
1. A【这是答案解析】
2. B【这是答案解析】

判断题
1. 正确【这是答案解析】
2. 错误【这是答案解析】

综合探究题
1. 答：参考答案内容
```

## 测试步骤

1. 准备测试文档
   - 创建一个Word文档(.docx格式)
   - 按照上述格式添加测试题目

2. 访问管理后台
   - 打开浏览器访问：http://localhost:5000/admin
   - 点击"题库管理"标签

3. 导入题目
   - 点击"导入题目"按钮
   - 选择科目（如：信息技术）
   - 上传Word文档
   - 点击"开始导入"

4. 查看结果
   - 导入成功后，会显示成功导入的题目数量
   - 可以在"题目列表"中查看导入的题目（功能开发中）

## 注意事项

1. 文件格式必须是.docx（不支持.doc）
2. 题目编号必须从1开始，连续编号
3. "参考答案"分隔符必须与文档中的完全一致
4. 答案中的题型标题必须与题目中的标题一致
5. 选项必须是A、B、C、D格式
6. 答案中的括号【】是可选的，用于添加解析

## 故障排查

### 问题：未解析到任何题目
- 检查文件格式是否为.docx
- 检查文档中是否有题目编号（如"1."、"1、"）
- 检查文档是否为空

### 问题：部分题目答案未匹配
- 检查"参考答案"分隔符是否正确
- 检查答案部分的题型标题是否正确
- 检查题目编号是否连续

### 问题：导入失败
- 检查网络连接
- 检查服务器日志
- 确认数据库连接正常

## 数据库说明

科目数据已初始化：
- 1: 信息技术
- 2: 通用技术
- 3: 美术
- 4: 音乐
- 5: 综合实践

导入的题目会保存到数据库的questions表中，包含以下字段：
- subject_id: 科目ID
- type: 题型（single_choice/judge/essay）
- question_text: 题目内容
- options: 选项（JSON数组，仅单项选择题）
- answer: 答案
- explanation: 答案解析
- sort_order: 排序顺序
