# 管理后台访问说明

## 服务状态
✅ 开发服务器运行正常 - 端口 5000
✅ Admin页面已加载 - http://localhost:5000/admin
✅ 数据库连接正常

## 访问方式

### 方法1：直接访问
在浏览器中打开：
```
http://localhost:5000/admin
```

### 方法2：如果无法访问，尝试以下操作

1. **清除浏览器缓存**
   - Chrome/Edge: Ctrl+Shift+Delete (Windows) 或 Cmd+Shift+Delete (Mac)
   - 选择"缓存的图片和文件"
   - 点击"清除数据"
   - 重新访问 http://localhost:5000/admin

2. **使用无痕/隐私模式**
   - Chrome/Edge: Ctrl+Shift+N (Windows) 或 Cmd+Shift+N (Mac)
   - 在无痕窗口中访问 http://localhost:5000/admin

3. **检查端口占用**
   在终端执行：
   ```bash
   curl http://localhost:5000/admin
   ```
   如果看到HTML内容，说明服务正常。

4. **重启浏览器**
   - 完全关闭浏览器
   - 重新打开并访问

## 页面功能说明

访问成功后，你会看到：

### 顶部导航
```
桂·职测助手 - 后台管理系统
```

### 左侧菜单
- 📚 题库管理
- 📝 模拟卷管理
- 🔑 VIP码管理
- 📦 订单管理
- 📰 资讯管理

### 题库管理功能
1. 点击"导入题目"按钮
2. 选择科目（信息技术、通用技术、美术、音乐、综合实践）
3. 上传Word文档(.docx格式)
4. 点击"开始导入"

## 测试用Word文档

你可以创建一个简单的测试文档，内容如下：

```
单项选择题
1. 1+1等于多少？
A. 1
B. 2
C. 3
D. 4

判断题
1. 地球是圆的。

参考答案
单项选择题
1. B【这是一加一等于二】

判断题
1. 正确【地球是圆的】
```

保存为test.docx，然后导入测试。

## 如果仍然无法访问

请尝试：
1. 使用不同浏览器
2. 检查是否有防火墙或代理拦截
3. 确认在本地网络环境（不是远程访问）
